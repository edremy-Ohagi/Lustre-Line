# Brand Assets

This folder stores source brand assets for 南望玉叙.

## Current Assets

- `logo.png`：user-provided logo source file
- `logo_transparent_bg.png`：transparent-background logo source, preferred for frontend display

## Frontend Copies

The web app keeps runtime-ready copies under:

```text
apps/web/public/brand/
```

Current generated copies:

- `logo.png`：full transparent logo for page-level or future footer usage
- `logo-mark.png`：cropped transparent emblem for compact header usage

Frontend projects may later copy or reference optimized versions from this folder, depending on the chosen build setup.
