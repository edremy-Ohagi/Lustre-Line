# Product Decisions

## Positioning

南望玉叙 is a real jewelry showcase and consultation project for domestic customers.

The first release is not a full e-commerce system. It should focus on:

```text
品牌展示
商品展示
商品详情
多渠道咨询转化
```

## Confirmed Decisions

| Item | Decision |
| --- | --- |
| Brand name | 南望玉叙 |
| Customer market | 国内客户为主 |
| Language | 中文为主 |
| Visual direction | 轻奢温柔 |
| Product categories | 项链、手链、耳钉、戒指 |
| Product images | 第一版使用高质量占位图 |
| Contact channels | 需要微信 / 小红书二维码，也可展示多个联系方式 |
| Price display | 统一人民币展示，允许“询问价格” |
| Product labels | 新品、推荐、定制 |
| Product details | 弹窗预览 + 可跳转独立详情页 |
| Hero direction | 商品展示和咨询转化优先 |
| SEO | 第一阶段不重点做 |
| Domain | 暂无 |
| Payment | 后续大概率接国内支付，MVP 不做 |

## Product Principle

First make the site good enough for the founder to review and show to potential customers. Then add content management and commerce features.
