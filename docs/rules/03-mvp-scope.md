# MVP Scope

## MVP Goal

Create a customer-facing front-end MVP that can be shared with the founder through GitHub Pages.

## Must Have

- Mobile-first landing page
- Brand header with logo / brand name
- Hero section focused on product display and consultation conversion
- Product categories:
  - 项链
  - 手链
  - 耳钉
  - 戒指
- Product list
- Product labels:
  - 新品
  - 推荐
  - 定制
- RMB price display
- Support for 询问价格
- Product detail modal
- Link from modal to independent product detail page
- WeChat QR code area
- Xiaohongshu QR code area
- Multiple contact channels
- High-quality placeholder product images
- GitHub Pages-friendly deployment

## Should Have

- Category filtering
- Featured product section
- Consultation CTA near hero and product details
- Copy button for WeChat ID if a WeChat ID is provided later
- Clear empty / placeholder states for missing QR codes

## Not In MVP

- Admin backend
- Database
- Java backend
- Login
- Order system
- Payment
- Inventory
- Membership
- Coupons
- Deep SEO work

## Future Phases

After MVP confirmation:

1. Content management backend
2. Java Spring Boot API
3. PostgreSQL persistence
4. Image upload and storage
5. DigitalOcean deployment
6. Domestic payment integration
