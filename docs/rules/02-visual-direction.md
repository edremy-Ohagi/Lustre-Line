# Visual Direction

## Core Vibe

```text
轻奢
温柔
东方感
细腻
留白
柔和光泽
适合送礼和日常佩戴
```

## Visual Priorities

- Jewelry images should be the main visual signal.
- The first viewport should clearly show the brand and product atmosphere.
- The page should feel refined and warm, not like a generic online store.
- Text should be short, clear, and sales-aware.
- Use whitespace to create calmness, but avoid emptiness on mobile.

## Suggested Palette Direction

- Warm white
- Soft jade green
- Muted gold
- Gentle blush or clay accent
- Deep ink text

## Avoid

- Overly promotional e-commerce style
- Loud red and gold luxury cliches
- Dark black-gold nightclub style
- Tech dashboard feeling
- Excessive gradients
- Decorative blobs or irrelevant ornaments
- Heavy animations that distract from products

## Logo Rule

Use the provided logo from:

```text
assets/brand/logo.png
assets/brand/logo_transparent_bg.png
```

For web display, prefer the transparent-background version. If the full logo does not fit a compact mobile layout, use the cropped logo mark in the header and keep a text fallback.
